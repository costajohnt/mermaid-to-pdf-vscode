# How to Eat an Apple

## Apple Selection

```mermaid
flowchart TD
    A[Find Apple] --> B{Firm?}
    B -->|Yes| C{Smooth skin?}
    B -->|No| D[Find another]
    C -->|Yes| E[Good to eat!]

```

## Preparation

```mermaid
flowchart LR
    A[Wash Apple] --> B[Dry]
    B --> C[Remove sticker]
    C --> D[Ready!]
```

## Eating Process

```mermaid
sequenceDiagram
    participant You
    participant Apple
    
    You->>Apple: Pick up
    You->>Apple: Bite
    You->>Apple: Chew
    You->>Apple: Swallow
    You->>Apple: Repeat
```

## Benefits

- **Immediate**: Fiber and energy
- **Short-term**: Vitamins and antioxidants  
- **Long-term**: Heart health and digestion

## Tips

1. **Spiral Method**: Rotate while eating
2. **Section Method**: Eat in quarters
3. Always wash first
4. Eat around the core

Remember: "An apple a day keeps the doctor away!"
